Code Used

Bootstrap

This website uses Bootstrap as the base of all of the pages. Every site that I make uses Bootstrap (vanilla or modified) as the base code.

sNet

sNet is used on this website to provide a way to link 4 websites together, and make them seamless and to act as one.

Web Font API

On the MCFS page, you may have noticed that it loads a cursive font. This is thanks to Google Web Fonts API for loading these custom fonts.

StormEngine

StormEngine is a upcoming "web plugin" release. It incorporates GetPage, NoCSS, QuickPages, UniNav, sNet, WebFonts, Bootstrap, and more. All at the cost of free. Coming Soon! (hopefully)
