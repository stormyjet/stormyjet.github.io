NoCSS

My CSS Docs are equipped with NoCSS coding. To have this, just ensure you have the CSS Doc bootstrap.min.css declared. Then add a p with the class NoCSS that will display a message when the CSS does not load.

Using my CSS

All of my CSS will be at css/{css file}. When editing your code, ensure you have these going to the right place. Warning | Never link to CSS like this: stormyjet.com/css/{cssfile}. It increases load times.
